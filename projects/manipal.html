Manipal University – Homepage UX Redesign

A strategic UX redesign exploration focused on improving homepage clarity, information architecture, and program discovery for prospective students.

Role: Product Designer
Type: Concept Redesign
Scope: Homepage UX
Focus: IA & Conversion Optimization
Tools: Figma, Wireframing, Prototyping

Project Context

The homepage serves multiple user groups — prospective students, parents, international applicants, and alumni. Over time, content expansion often leads to clutter and diluted hierarchy.

This redesign focused on restructuring the homepage to reduce cognitive overload while improving discoverability and decision clarity.

Problems Identified

  • Dense content blocks competing for attention
  • Multiple CTAs without visual prioritization
  • Scattered program discovery pathways
  • Repetitive trust signals without clear grouping
  • Limited progressive disclosure

Redesign Goals

  • Establish stronger visual hierarchy
  • Improve scannability
  • Streamline program discovery
  • Strengthen primary CTAs
  • Reduce cognitive load

Information Architecture Shift

Before: Content-heavy layout

After: User-intent driven flow

Hero → Quick Access → Programs → Trust Signals → Research → Global → Testimonials → Final CTA

Redesigned Homepage (Concept)

Key UX Improvements

  • Clear CTA hierarchy (Primary + Secondary)
  • Task-based quick access cards
  • Filter-based program discovery
  • Structured grouping of trust signals
  • Improved whitespace and visual rhythm

Impact of the Redesign

Clearer

Primary CTA hierarchy with focused user pathways

Faster

Improved program discovery through structured grouping

Scannable

Reduced cognitive overload with better whitespace and layout rhythm
